# agent-chat-bot-ecommerce
Agent chat bot e-commerce with integration Google Sheets

Agente de IA Autônomo para Atendimento ao Cliente com Integração ERP/CRM (RAG)".

Orquestração de LLMs Avançada: Uso do nó AI Agent no modo Tools/Function Calling em vez de correntes lineares simples.

Gestão de Estado e Memória: Implementação de memória contextual (Window Buffer Memory) isolada por ID de sessão (sessionId) para suportar múltiplos usuários simultâneos.

Tomada de Decisão Autônoma (Function Calling): Capacidade da IA de identificar a intenção do usuário (Intent Recognition) e acionar a API do Google Sheets dinamicamente apenas quando necessário.
